InMAP formulation

  • Emission
  • Advection + Mixing
  • Reaction
  • Deposition
  • Exposure + Health Effects

Emission

  • "VOC", "NOx", "NH3", "SOx", and "PM2_5"
  • Shapefile format
  • Annual total
  • Can include stack "height", "diam", "temp", and "velocity" [m, m, K, and m/s].

Advection + Mixing

  • Annual average wind speeds
  • Parameters for wind "meandering" and sub-grid mixing

Reaction

  • InMAP only considers chemistry related to PM2.5 (no O3)
  • NH3 ⇆ particulate NH4
  • NOx ⇆ particulate NO3
  • VOC ⇆ SOA
  • SOx → particulate SO4
  • Primary PM2.5 stays that way

Deposition

  • Dry deposition (collisions with surfaces)
  • Wet deposition (absorption into clouds + droplet scavenging)

Exposure + Health Effects

  • Concentration changes
  • Population
  • Baseline mortality rate

					[OutputVariables]
					  TotalPM25 = "PrimaryPM25 + pNH4 + pSO4 + pNO3 + SOA"
					  DeathsAll = "coxHazard(loglogRR(totalpm25), POP, mota_rate*100)"

Using the terminal

  1. Open a terminal window:
    • Mac: Applications > Utilities > Terminal
    • Windows 10: Push 'Windows Key' > Type 'cmd.exe' > Push 'Enter'
  2. Type 'cd ' (i.e., 'c', 'd', 'space key').
  3. Click on the (unzipped) folder containing the tutorial data and drag it into the terminal window.
  4. Push the 'Enter' key.
  5. Start InMAP:
    • Mac: Type './inmap1.4darwin-amd64' (note the './') and push 'Enter'.
    • Windows 10: Type 'inmap1.4windows-amd64.exe' (with no './') and push 'Enter'.
  6. A user interface window should open in your browser.

Running the GEOS-Chem preprocessor

  • GUI
    1. Type 'preprocess_config.toml' into the '--config' input field.
    2. Click the 'Execute' button.
  • Command line: 
    inmap1.4XXXX preproc --config=preprocess_config.toml
    (where 'XXXX' denotes the program version for your operating system).
  • Note: In order for this to actually work, you would need to download the GEOS-Chem data. More information is available in the 'geos-chem/README.md'.
  • For now, we will use previously the previously created preprocessor output file at 'geos-chem/inmap_inputs_geoschem4x5.ncf'.

Creating emissions inputs

  • We will create example emissions scenarios using OpenStreetMap data for Beijing.
  • For example, we can compare whether a unit of emissions of primary PM2.5 is more damaging per mile of major roadways vs. residential roadways.
  • To do this we create two emissions input shapefiles by running the commands: 
    ./emissions/process_emissions_xxx -osmfile=emissions/beijing_china.osm.pbf -tag=highway -value=primary -pm25rate=1000 -out=run/primary_hw_emissions.shp
    and 
    ./emissions/process_emissions_xxx -osmfile=emissions/beijing_china.osm.pbf -tag=highway -value=residential -pm25rate=1000 -out=run/residential_hw_emissions.shp
  • More information is available in the emissions directory.

Running InMAP

  • GUI:
    1. Type "run_config.toml" in the --config input window.
    2. Select the "run" and "steady" subcommands.
    3. Set the '--EmissionShapefiles' field to '[emissions/primary_hw_emissions.shp]'.
    4. Set the '--OutputFile' field to 'output/primary_hw.shp'.
    5. Click "Execute".
  • Command line: 
    inmap1.4XXXX run steady --config=run_config.toml --EmissionShapefiles='[emissions/primary_hw_emissions.shp]' --OutputFile='output/primary_hw.shp'
  • ...and repeat for the residential roads.
  • Compare the output of the two simulations in QGIS
